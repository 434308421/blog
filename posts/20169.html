博客搭建教程

置顶 | 精品 | 发表于 2020-05-11 | 更新于: 2020-05-12 | 分类于 技术汇总 | | 浏览: 字数: 2.8k | 时长 ≈ 11 分钟

注：这是一份很详细的关于Next5主题的博客搭建教程

第一步 安装Git Bash

Git Bash下载地址

可以通过git version 查看当前版本 😁

第二步 安装NodeJs

NodeJs下载地址

注：LTS为长期支持版，Current为当前最新版

注：npm config set prefix “D:\Program Files\nodejs\node_global” 设置全局模块存放
注：npm config set cache “D:\Program Files\nodejs\node_cache” 设置全局cache存放

在命令行输入以下命令安装express（注：“-g”表示安装到global目录下，就是上面设置的node_global中）

npm install express -g

可以看到node_global/node_modules下有express了

安装cnpm，输入以下命令： npm install -g cnpm --registry=https://registry.npm.taobao.org

可以用过node -v查看当前版本

第三步 安装hexo

通过全局安装将hexo安装到node.js下

npm i -g hexo

可以用过hexo -v查看当前版本

然后再目标盘符内新建一个用于存放博客的文件

然后进入到文件内，鼠标右键，打开Git Bash Here

初始化命令：hexo init，初始化完成后会看到以下文件内容

解释一下：

node_modules：是依赖包
public：存放的是生成的页面
scaffolds：命令生成文章等的模板
source：用命令创建的各种文章
themes：主题
_config.yml：整个博客的配置
db.json：source解析所得到的
package.json：项目所需模块项目的配置信息

第四步 搭桥到GitHub

1.新建一个公共仓库

回到GitHub，配置GitHub账户信息

git config --global user.name "你的GitHub名字"
git config --global user.email "你的邮箱"

¶创建SSH

在GitBash输入：ssh-keygen -t rsa -C "youremail@example.com，生成ssh

然后输入找到cat id_rsa.pub查看文件中的SSH内容并复制

回到GitHub中，点击右上角头像，Setting,SSH and GPG keys,new SSH key，title随便取，key就填刚刚复制的那一段

在gitbash中验证是否添加成功：ssh -T git@github.com

第五步 本地预览并推到GitHub

¶本地预览

用编辑器打开你的blog项目，修改_config.yml文件的一些配置(冒号之后都是有一个半角空格的)：

deploy:
type: git
repo: git@github.com:434308421/434308421.github.io.git
branch: master

回到gitbash中，进入你的blog目录，分别执行以下命令：

hexo clean
hexo generate
hexo server

注:hexo 3.0把服务器独立成个别模块，需要单独安装：
安装server：npm install hexo-server --save

打开浏览器输入：http://localhost:4000查看效果

¶上传到GitHub

安装deployer：npm install hexo-deployer-git --save

这样才能将你写好的文章部署到github服务器上并让别人浏览到

不安装的话无法使用hexo d

执行命令(建议每次都按照如下步骤部署)：

hexo clean
hexo generate
hexo deploy

注意deploy的过程中要输入你的username及passward

在浏览器中输入https://434308421.github.io/就可以看到你的个人博客了

感觉gitbash中东西太多的时候输入clear命令清空

第六步 配置主题

hexo初始化之后默认的主题是landscape , 然后你可以去github中搜索你要的主题名称，里面都会有该主题的如何使用的介绍，按着来就好了，反正就是改改改！我选的是基于Next5主题美化的一款hexo博客主题,看起来挺不错，至少是我喜欢的类型。

Next5博客主题：https://bestzuo.cn
Next5博客地址：https://github.com/Sanarous/hexo-theme-next5-polished

基于Next5魔改的主题：https://www.liaofuzhan.com
基于Next5魔改的主题地址：https://github.com/leafjame/hexo-theme-next5-leaf

找到需要的主题GitHub仓库地址在git bash中执行：git clone 主题地址

然后修改主题文件夹的名字，再修改博客根目录_config.yml里面的theme配置主题名称

最后：hexo clean && hexo generate && hexo server预览

或者：hexo clean && hexo generate && hexo d上传GitHub

以上内容来源于此地

¶文章元数据设置

元数据就是显示在 home 页的文章创建于、更新于、阅读次数之类的数据，搜索 post_meta：

post_meta:
item_text: true # 是否显示对应的文字
created_at: true # 是否显示 创建于
updated_at: # 是否显示 更新于
enabled: false
# 更新日期显示规则，只有更新日期与创建日期不同时，才会显示
another_day: true
categories: true # 是否显示分类信息

¶文章字数统计设置

在站点根目录打开 git bash，输入npm i --save hexo-wordcount

进入主题配置文件 _config.yml，修改如下配置：

Post wordcount display settings
Dependencies: https://github.com/willin/hexo-wordcount
文章字数展示设置
post_wordcount:

文本显示

item_text: true

文章字数统计

wordcount: true

阅读时长

min2read: true

站点总字数统计

totalcount: false

该post_wordcount的所有设置另起一行显示

separated_meta: true

¶开启相关文章推荐功能

文件位置：D:\临时文件夹\myblog\blog\themes\next5_config.xml

搜索 related_posts，找到如下配置：

related_posts:
enable: true # 是否开启
title: 相关文章推荐 # 标题
display_in_home: false # 是否在首页显示，建议为false
params:
maxCount: 5 # 相关文章的最大数量
#PPMixingRate: 0.0
#isDate: false
#isImage: false
#isExcerpt: false

开启相关文章推荐需要安装hexo-related-popular-posts模块，即在 hexo站点根目录下使用npm install hexo-related-popular-posts --save安装模块，然后开启上面的相关文章功能就可以啦~
